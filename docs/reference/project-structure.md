## Root Level Files

### Program.cs
Application startup and configuration:
- Dependency injection setup
- Middleware configuration
- Service registration
- Logging setup

### AppConstants.cs
All application constants in one place:
- Server URLs and ports
- File size limits
- Configuration values
- Logging event IDs

### ReportBuilder.sln
Visual Studio solution file - contains project references.

### ReportBuilder.Web.csproj
C# project file:
- NuGet package references
- Build configuration
- Target framework (.NET 9.0)

### build-distribution.ps1
PowerShell script to build distribution:
- Cleans previous builds
- Restores packages
- Publishes self-contained app
- Creates ZIP archive

### mkdocs.yml
MkDocs configuration:
- Site settings
- Theme configuration
- Documentation structure
- Extensions

### .gitignore
Files excluded from version control:
- bin/, obj/ (build output)
- publish/ (release builds)
- .vs/ (VS cache)
- node_modules/ (if used)

### .editorconfig
Code style rules (in Config/):
- Indentation (4 spaces)
- Line endings (CRLF)
- Charset (UTF-8)
- File types and rules
