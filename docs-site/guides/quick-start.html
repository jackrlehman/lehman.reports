    Quick Start Guide¶

    Get up and running in 5 minutes.

    Prerequisites¶

    • .NET 9.0 SDK or later
    • A modern web browser
    • Windows 10/11 or macOS/Linux

    Installation¶

    Step 1: Download and Install .NET¶

    Visit: https://dotnet.microsoft.com/download

    Download .NET 9.0 SDK (not Runtime)

    Step 2: Verify Installation¶

    Open PowerShell or Terminal and run:

    dotnet --version

    You should see: 9.0.x or higher

    Running the Application¶

    Development Mode (Recommended)¶

    Auto-reloads when you make changes:

    cd C:\git\lehman.reports
dotnet watch run

    Single Run¶

    cd C:\git\lehman.reports
dotnet run

    Access the Application¶

    The app automatically opens in your browser at:

    http://localhost:5000

    If it doesn't open automatically: - Wait 10-15 seconds for the server to start - Manually navigate to http://localhost:5000

    Creating Your First Report¶

    1. Click "Mobile App Store Performance Report" in the menu
    2. Fill in your company name and dates
    3. Toggle sections you want to include
    4. Fill in the metrics for each section
    5. Click "Generate PDF Report" to download

    Troubleshooting¶

    Port 5000 Already in Use¶

    Error: Address already in use

    Solution: 

    # Kill existing dotnet process
taskkill /F /IM dotnet.exe

# Try again
dotnet watch run

    Connection Refused¶

    Error: Connection refused or blank page

    Solution: - Wait 10-15 seconds (first start takes time) - Refresh browser page - Check PowerShell for error messages

    Large Build Time¶

    First run takes longer (2-3 minutes) because .NET is downloading and compiling everything. This is normal.
